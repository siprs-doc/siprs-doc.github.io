7.函数解析

引擎提供对于复杂json结构体的参数求解功能。

Fn::GetAtt

从一个json结构体中，取某一个参数

{“processInstanceId” : 
    {
          "Fn::GetAtt": [
            {
              "createProcessInstanceId" : "111",
              "name":"jason"
            },
            "createProcessInstanceId"
          ]
        }
}
等同于
{"processInstanceId" : "111"}
  • 参数1：json结构体
  • 参数2：取参数的key

如上所示，为取出json结构体中的createProcessInstanceId值。

Fn::GetProcessOutputs

从一个已有的流程实例中获取某一个输出结果

{
    "DefaultValue": {
      "Fn::GetProcessOutputs": [
        “111”,
        "createTomcat01",
        {
          "Exclude": [
            "instanceCode",
            "instanceId",
            "privateIp",
            "name"
          ]
        }
      ]
    }
}
